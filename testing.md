KhedutSathi ai powered comprehensive platform for Indian farmers providing AI-based crop recommendations, weather forecasts, market prices, and farming guidance.

Key Features
Crop Recommendation: Uses AI to suggest the best crops based on soil type, weather, and market demand
Weather Forecast: Provides hyper-local weather updates and alerts
Market Prices: Real-time mandi prices from across India
Farming Guidance: AI-powered chatbot and expert advice for farming queries
Marketplace: Buy/sell equipment, seeds, and other farming products
Technology Stack
Frontend: React, Vite, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB, Redis
AI/ML: TensorFlow.js, scikit-learn (simulated via API)
Deployment
Frontend: Netlify (vercel-integrated)
Backend: Render
AI Model: Cloud-hosted TensorFlow model
Running Locally
1. Backend
cd backend
npm install
npm start
2. Frontend
cd frontend
npm install
npm run dev
Access the app at http://localhost:5173
